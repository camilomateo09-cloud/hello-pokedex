# Pokédex

Este proyecto consiste en el desarrollo de una **Pokédex interactiva** inspirada en la aplicación oficial de Pokémon.  
La aplicación permite visualizar un listado de Pokémon y acceder a una **pantalla de detalle individual** para cada uno, mostrando información como tipos, estadísticas base, peso, altura y descripción.

El proyecto está desarrollado utilizando **HTML, CSS y JavaScript**, consumiendo datos reales desde la **API**.

---

## Objetivo del proyecto

El objetivo principal de este proyecto es:

- Practicar el desarrollo frontend con HTML, CSS y JavaScript.
- Aprender a consumir una API externa (PokéAPI).
- Implementar navegación entre pantallas simulando el comportamiento de una aplicación móvil.

---

## Funcionalidades

- Listado de Pokémon en formato grid (3x3).
- Barra de búsqueda.
- Vista de detalle por Pokémon
